# Hypothesis Guard: Social Science Domain

## 何時載入
當研究假設涉及心理學、社會學、教育學、政治學、經濟學、人類行為研究時。

---

## Social Science 假設的特殊要求

### 1. 構念效度 (Construct Validity)
```
□ 假設中的核心構念是否有明確的操作型定義？
  - "焦慮" → 用什麼量表？（GAD-7? STAI? 自行開發？）
  - "社經地位" → 用收入、教育、職業、還是綜合指標？
  - "創造力" → 發散性思維測驗？專家評分？自我報告？
□ 操作型定義是否被該領域廣泛接受？
□ 是否有多重測量方法（triangulation）？
□ 是否討論了構念與測量之間的差距？
```

### 2. 再現性危機意識
```
社會科學正在經歷再現性危機。假設生成時必須考慮：

□ 相同或類似假設是否已有再現性研究？
  - 搜索 "[主題] replication" 和 "[主題] reproducibility"
□ 原始效果量是否可能被誇大？
  - 社科領域的 published effect sizes 平均比 replication 大 50%
  - 對文獻中的效果量打折：假設實際效果是報告值的 50-70%
□ 樣本是否有 WEIRD 偏誤？
  - WEIRD = Western, Educated, Industrialized, Rich, Democratic
  - 大部分心理學研究基於 WEIRD 樣本
  - 假設如果基於 WEIRD 樣本的文獻 → 標記泛化限制
□ 是否有預註冊計畫？
  - OSF (osf.io) 或 AsPredicted
  - 社科領域強烈建議預註冊以避免 HARKing
```

### 3. 文化敏感性
```
□ 假設是否有文化特異性？
  - 在文化 A 成立的關係，在文化 B 可能不成立
  - 例：個人主義 vs. 集體主義文化中的動機理論
□ 測量工具是否經過跨文化驗證？
  - 翻譯 ≠ 驗證（需要獨立的心理計量驗證）
□ 是否考慮了語言和脈絡差異？
```

### 4. 社科假設常見陷阱
```
⚠ 測量反應性
  測量行為本身會改變被測量的行為
  例：知道被觀察 → 改變行為（Hawthorne 效應）
  修正：討論反應性威脅和控制策略

⚠ 自我報告偏誤
  參與者可能基於社會期望回答，而非真實態度
  例：種族偏見、性行為、健康行為的自我報告
  修正：使用間接測量或行為指標補充自我報告

⚠ 生態效度
  實驗室中的發現可能不適用於真實生活情境
  修正：討論實驗室-真實世界的差距

⚠ 第三變項
  社科中的因果推論極其困難，幾乎總有未測量的混淆變項
  修正：列出至少 3 個潛在的混淆變項

⚠ 時序性
  橫斷面研究無法確立因果方向
  例："社交媒體使用 ↔ 憂鬱" 方向不確定
  修正：明確標記假設需要縱貫性設計才能驗證因果

⚠ 效果量虛胖
  小樣本 + publication bias → 文獻中的效果量可能被誇大
  修正：Power analysis 使用保守的效果量估計
```

### 5. 搜索策略
```
必查資料庫：
- PsycINFO / PsycARTICLES（心理學）
- ERIC（教育學）
- SSRN（社會科學預印本）
- Google Scholar（跨領域）
- Cochrane / Campbell Collaboration（systematic reviews）

特殊搜索：
- "[主題] meta-analysis" → 效果量的最佳估計
- "[主題] replication" → 再現性結果
- "[量表名] validation" → 測量工具的效度證據
- "[主題] cultural differences" → 跨文化證據
```

---

## 輸出格式
```
## Social Science 假設驗證補充報告
- 構念定義：[構念] → [操作型定義] — [效度等級]
- 再現性狀態：[原始研究已被再現/未被再現/未嘗試再現]
- WEIRD 風險：[HIGH/MEDIUM/LOW]
- 效果量估計：文獻報告 [d/r] → 保守估計 [d/r × 0.6]
- 預註冊建議：[是否需要 / 平台建議]
- 文化特異性：[普遍性/文化特異/未確認]
```
