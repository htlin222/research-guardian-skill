# Literature Guard: Social Science Citation Standards

## 何時載入
當文獻回顧涉及心理學、社會學、教育學、經濟學文獻時。

---

## 再現性狀態標記（必做）
```
□ 關鍵引用是否被成功再現過？
  搜索 "[效應名] replication"
  - 已再現 → 信心加成
  - 再現失敗 → 信心大幅降低 ⚠
  - 未嘗試 → 標記 "unreplicated"

已知再現失敗（引用時需特別謹慎）：
  Ego depletion, Power posing, Stanford Prison Experiment,
  Facial feedback hypothesis
```

## 效果量校準
```
□ 社科 publication bias → 發表的效果量平均被誇大 ~50%
  規則：引用效果量時附帶折扣
□ 優先引用 meta-analysis 的效果量
□ 檢查 meta-analysis 中的 publication bias 分析
  （funnel plot, Egger's test）
```

## 量表與測量工具引用
```
□ 引用原始開發論文？
□ α > .70 的效度/信度驗證？
□ 跨文化使用時引用該文化的驗證研究？
□ 量表是否有更新版本？（DSM-IV → DSM-5）
```

## 抽樣偏誤標記
```
□ WEIRD 樣本標記？
□ 線上招募（MTurk/Prolific）標記？
□ 學生樣本 vs. 一般人群區分？
```

## 驗證工具
```
PsycINFO, ERIC, SSRN, Curate Science (curatescience.org)
```

## 新近性標準
```
Empirical findings:    最近 10 年
Theoretical frameworks: 可較舊（需確認修訂版）
Measurement tools:     最新驗證版本
Meta-analyses:         最近 5 年
```

## 輸出格式
```
## Social Science 文獻驗證補充
- 再現性：[N] 已再現 / [N] 失敗 / [N] 未嘗試
- WEIRD 比例：[N]%
- 效果量校準：原始 [d] → 保守估計 [d × 0.6]
```
