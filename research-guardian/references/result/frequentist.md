# Result Guard: Frequentist Statistics

## 何時載入
當結果報告使用頻率學派統計方法（t-test, ANOVA, 迴歸, 卡方等）時。

---

## 報告規範檢查

### p 值報告
```
□ 報告精確 p 值？（p = .023，不是 p < .05）
□ 避免 p 值的錯誤解讀？
  ❌ "95% 機率假設為真"
  ❌ "5% 機率結果是偶然"
  ✅ "在虛無假設下觀察到此數據的機率為 p"
□ p = .05 附近（.04-.06）是否特別標注？（fragile significance）
□ 多重比較是否校正？
  - Bonferroni（最保守）
  - Holm-Bonferroni（較少犧牲效力）
  - Benjamini-Hochberg FDR（控制偽發現率）
  - 比較次數 N → 報告校正方法和校正後 p 值
```

### 效果量（必做）
```
□ 是否報告了效果量？（只有 p 值是不夠的）
  - 組間差異：Cohen's d, Hedges' g
  - 相關性：Pearson r, Spearman ρ
  - 變異解釋量：R², η², ω², f²
  - 比率：OR, RR, HR
□ 效果量解讀基準：
  Cohen's d:  small=0.2, medium=0.5, large=0.8
  r:          small=0.1, medium=0.3, large=0.5
  η²:         small=0.01, medium=0.06, large=0.14
□ 效果量是否有信賴區間？
```

### 信賴區間
```
□ 主要結果是否報告 95% CI？
□ CI 的解讀是否正確？
  ❌ "真實值有 95% 的機率落在這個區間"
  ✅ "如果重複抽樣 100 次，約 95 次的 CI 會包含真實值"
□ CI 是否跨越零（或臨床無意義區間）？
  - CI 包含零 → 效果可能不存在
  - CI 跨越 MCID → 臨床意義不確定
```

### 各檢定的報告格式
```
t-test:    t(df) = X.XX, p = .XXX, d = X.XX, 95% CI [X.XX, X.XX]
ANOVA:     F(df1, df2) = X.XX, p = .XXX, η² = .XX
迴歸:      β = X.XX, SE = X.XX, t = X.XX, p = .XXX, R² = .XX
卡方:      χ²(df, N = XX) = X.XX, p = .XXX, Cramér's V = .XX
相關:      r(N) = .XX, p = .XXX, 95% CI [.XX, .XX]
Mann-Whitney: U = XXXX, z = X.XX, p = .XXX, r = .XX
```

### 常見統計錯誤偵測
```
⚠ 自由度與樣本量不匹配
  注意：df < N-2 不一定是錯誤。合法情況包括：
    - 分析只比較了兩個子群體（排除中間組）
    - 配對設計（df = pairs - 1）
    - 多層模型（df 取決於群組數而非個體數）
  判定規則：如果分析明確聲明了子集選擇 → WARN 不是 FAIL
            如果未說明為什麼 df ≠ N-2 → FAIL
  t(48) 但報告 N = 55 → 某些資料被排除但未說明

⚠ F 值或 t 值為負
  F 值永遠非負，t 值可正可負 → F < 0 是計算錯誤

⚠ p 值與檢定統計量不一致
  t(20) = 1.5 不可能 p < .001 → 數字有誤

⚠ 效果量異常大
  d > 2.0 在社科中極罕見 → 需要解釋

⚠ 所有 p 值恰好為 .000 或 < .001

⚠ p 值分佈異常窄
  如果報告了 N > 10 個 p 值：
  - 全部落在同一個 0.04 寬的區間（如 .01-.04）→ 🔴 不自然
  - 真實的 p 值分佈應該更分散（有些 < .001，有些 > .10）
  - 所有 p 值都「恰好」顯著（.01-.049）→ 高度可疑（p-hacking 信號）
  - 規則：計算 p 值的 range（max - min），如果 range < 0.04 且 N > 10 → 🔴
  可能是軟體輸出直接貼上，未報告精確值
```

## 輸出格式
```
## Frequentist 統計驗證
- p 值報告：[精確/分級/有誤]
- 效果量報告：[有/無] — [大小合理性]
- CI 報告：[有/無]
- 格式合規：[APA 合規/部分/不合規]
- 統計錯誤偵測：[N] 個可疑數值
```
