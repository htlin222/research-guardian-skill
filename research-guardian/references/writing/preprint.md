# Writing Guard: Preprint

## 何時載入
當撰寫預印本（arXiv, bioRxiv, medRxiv, SSRN 等）時。

---

## 預印本的特殊考量

### 品質標準不能因為「只是預印本」而降低
```
核心原則：預印本沒有同行審查，但讀者不會因此降低期望。
一篇品質差的預印本會永久損害作者聲譽。

□ 是否達到了可投稿至同行審查期刊的品質？
  - 如果不是 → 為什麼要發佈？
□ 是否在標題或摘要中標明這是預印本且未經同行審查？
□ 是否有版本控制計畫？
  - v1 → 收到反饋後 → v2 → 投稿至期刊
```

### 預印本特有風險
```
⚠ 優先權搶跑
  - 在確認結果穩健之前發佈 → 如果結果有誤會很尷尬
  - 修正：確保結果至少在內部被獨立驗證過

⚠ 媒體放大
  - 預印本可能被新聞媒體報導為「已證實的科學」
  - 修正：在摘要和結論中明確標示初步性質
  - 特別是醫學預印本 → 可能影響公共健康決策

⚠ 引用黏著性
  - 預印本被大量引用後，即使後來被修正，舊版本的結論已散播
  - 修正：如果後續版本有重大修改，在新版本中明確標示

⚠ Scooping 風險
  - 發佈預印本不等於完全的優先權保護
  - 某些期刊不承認預印本時間戳
  - 修正：了解目標期刊的預印本政策
```

### 預印本 vs. 正式論文的寫作差異
```
允許的放鬆：
  ✅ 更自由的結構（不必嚴格遵循 IMRAD）
  ✅ 更多的推測性討論（但需明確標記為推測）
  ✅ 負面結果或初步結果的報告
  ✅ 技術報告風格（更多實作細節）

不能放鬆的：
  ❌ 引用的準確性（幻覺引用在預印本中同樣不可接受）
  ❌ 數據的正確性
  ❌ 結論的適度性
  ❌ 倫理合規（IRB 等）
```

### 預印本平台選擇
```
- arXiv：物理、數學、CS、統計（需要 endorsement）
- bioRxiv：生物學
- medRxiv：醫學（有基本品質篩選）
- SSRN：社會科學、法律、經濟
- ChemRxiv：化學
- engrXiv：工程
- PsyArXiv：心理學

平台特有規則：
- medRxiv 不接受臨床試驗結果（需送正式期刊）
- arXiv 的 moderation 可能拒絕品質過低的投稿
```

### 版本管理
```
□ 版本號是否遞增？（v1, v2, v3）
□ 每個版本是否有 changelog？
□ 重大修改是否在新版本的開頭說明？
□ 最終投稿版本是否也更新到預印本平台？
```

## 輸出格式
```
## Preprint 寫作驗證補充
- 品質等級：[可投稿期刊 / 需改善 / 技術報告等級]
- 風險評估：[媒體放大風險 / Scooping 風險 / 引用黏著風險]
- 平台選擇：[推薦平台] — [理由]
- 版本控制：[有計畫 ✅ / 無計畫 ⚠]
```
